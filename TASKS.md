# macOS PDF Viewer 開發任務

## 第一階段：基礎建設

- [ ] **任務 1.1**: 建立新的 Xcode 專案。
    - 類型：macOS App
    - 框架：SwiftUI / AppKit
    - 語言：Swift
- [ ] **任務 1.2**: 整合 `PDFKit`。
- [ ] **任務 1.3**: 實作開啟 PDF 檔案功能。
    - 透過 `NSOpenPanel` 讓使用者選擇檔案。
    - 將選擇的 PDF 載入到 `PDFView` 中。

## 第二階段：核心功能開發

- [ ] **任務 2.1**: 實作單頁/雙頁瀏覽模式切換。
    - 在 UI 上新增一個切換按鈕。
    - 綁定按鈕狀態到 `PDFView` 的 `displayMode` 屬性 (`.singlePage` 或 `.twoUp`)。
- [ ] **任務 2.2**: 實作翻頁方向切換。
    - 在 UI 上新增一個切換按鈕。
    - 根據按鈕狀態設定 `PDFView` 的 `displaysAsBook` 屬性，並結合 `pageBreakMargins` 來控制左右翻頁的呈現。
    - 注意：`PDFKit` 對於右開本書籍（Right-to-Left）的支援可能需要額外處理。
- [ ] **任務 2.3**: 實作滑鼠滾輪/觸控板翻頁功能。
    - 監聽滑鼠滾輪與觸控板的捲動事件。
    - 將捲動事件轉換為 `goToNextPage` 或 `goToPreviousPage` 動作。

## 第三階段：UI/UX 優化

- [ ] **任務 3.1**: 設計並實作工具列。
    - 包含所有功能按鈕。
    - 顯示目前頁碼與總頁數。
- [ ] **任務 3.2**: 完善使用者體驗。
    - 確保視窗縮放時，PDF 內容能正確自適應。
    - 優化滾動與翻頁的流暢度。

## 第四階段：測試與發布

- [ ] **任務 4.1**: 功能測試。
    - 測試所有核心功能是否正常運作。
    - 使用不同大小與類型的 PDF 檔案進行測試。
- [ ] **任務 4.2**: 準備應用程式發布。
    - 設定 App Icon。
    - 處理程式碼簽署與封存 (Archiving)。
