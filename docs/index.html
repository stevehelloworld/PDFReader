隱私權政策 (Privacy Policy)

最後更新日期：2025 年 11 月 25 日

感謝您使用 PDFViewer（以下簡稱「本應用程式」）。本應用程式由 Steve Yeh 開發。我們非常重視您的隱私權，因此制定了本隱私權政策來說明我們如何處理您的資訊。

1. 資料收集與使用 (Data Collection and Use)

本應用程式不收集、不儲存、也不傳輸任何使用者的個人資料。

本應用程式是一個完全離線運行的 PDF 閱讀工具。所有的 PDF 檔案處理（包括開啟、閱讀、渲染）皆在您的裝置本機上完成。我們不會將您的文件上傳至任何伺服器。

2. 檔案存取權限 (File Access)

為了提供閱讀功能，本應用程式需要存取您選擇開啟的 PDF 檔案。此權限僅用於：

  • 顯示文件內容
  • 記錄閱讀進度（頁碼）

這些資訊僅儲存在您的裝置本機（UserDefaults）中，不會與開發者或第三方共享。

3. 第三方服務 (Third-Party Services)

本應用程式不包含任何第三方分析工具（如 Google Analytics、Firebase）或廣告服務，因此不會有第三方收集您的使用數據。

4. 政策變更 (Changes to This Policy)

我們可能會不時更新我們的隱私權政策。建議您定期查看本頁面以獲取任何更改。這些更改在發佈到本頁面後立即生效。

5. 聯絡我們 (Contact Us)

如果您對本隱私權政策有任何疑問或建議，請透過 GitHub Issues 聯絡我們：
https://github.com/stevehelloworld/PDFReader/issues
