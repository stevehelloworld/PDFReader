# macOS PDF Viewer 規格書

## 1. 總覽

本文件定義了一個 macOS PDF 瀏覽器應用程式的功能與規格，旨在提供使用者一個簡潔、高效的 PDF 閱讀體驗。

## 2. 核心功能

### 2.1. PDF 文件處理
- **開啟文件**: 使用者可以透過系統的檔案選擇器來開啟本地的 PDF 檔案。
- **文件渲染**: 清晰地渲染並顯示 PDF 內容。

### 2.2. 瀏覽模式切換
- **單頁模式 (Single Page View)**: 一次顯示單一頁面，適合垂直滾動閱讀。
- **雙頁模式 (Double Page View / Book Mode)**: 模擬書本，一次顯示兩個頁面，適合橫向翻頁閱讀。
- **模式切換**: 提供一個易於操作的 UI 控制項（例如：工具列按鈕），讓使用者可以隨時在單頁與雙頁模式之間切換。

### 2.3. 翻頁方向設定
- **由左至右 (Left-to-Right)**: 傳統的翻頁方式，下一頁在右邊。此為預設設定。
- **由右至左 (Right-to-Left)**: 適用於習慣右翻的書籍（例如：漫畫、古籍）。下一頁在左邊。
- **方向切換**: 提供一個 UI 控制項，讓使用者可以根據閱讀習慣選擇翻頁方向。

### 2.4. 互動方式
- **滑鼠滾輪/觸控板**: 使用者可以使用滑鼠滾輪的垂直滾動或觸控板的雙指滑動手勢來進行翻頁。

## 3. 使用者介面 (UI)

- **主視窗**: 顯示 PDF 內容的主要區域。
- **工具列**: 位於視窗頂部，包含以下控制項：
    - 開啟檔案按鈕
    - 瀏覽模式切換按鈕 (單頁/雙頁)
    - 翻頁方向切換按鈕 (左至右/右至左)
    - 頁碼顯示與跳轉功能。

## 4. 技術規格

- **平台**: macOS
- **開發框架**: Swift, AppKit/SwiftUI
- **核心渲染引擎**: PDFKit

## 5. 非功能性需求

- **效能**: 能夠流暢地開啟與捲動大型 PDF 檔案。
- **易用性**: 介面直觀，使用者無需學習即可上手。
